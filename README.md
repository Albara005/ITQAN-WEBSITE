# إتقان — ACADEMIC STUDIO

منصة الملخصات والخدمات الأكاديمية. موقع عربي (RTL) من صفحة واحدة مع لوحة تحكم للإدارة وباك-اند بسيط لاستقبال الطلبات.

## المميزات

- **هيرو سينمائي** — فيديو يتحرك إطارًا بإطار مع السكرول (يُرسم على `canvas` من صور في `frames/`).
- **أقسام**: الخدمات · طريقة العمل (٥ خطوات) · أعمالنا · آراء العملاء · الأسعار · الأسئلة الشائعة · التواصل.
- **نموذج طلب متعدد الخطوات**: اختيار الخدمة → رفع الملفات → الإضافات → بيانات العميل، مع رقم طلب متسلسل (`#1001`) وزر تأكيد عبر واتساب.
- **لوحة تحكم `/admin`** محمية بإيميل + كلمة مرور: عرض الطلبات وتغيير حالتها وتحميل ملفاتها، إدارة «نماذج الأعمال» (رفع صور + تعديل العناوين)، وضبط رقم الواتساب.
- متجاوب بالكامل مع جميع الأجهزة، ويحترم `prefers-reduced-motion`.

## التشغيل

```bash
npm install        # يثبّت busboy و nodemailer
node serve.cjs     # يشغّل الموقع + الباك-اند على http://localhost:8000
```

- الموقع: `http://localhost:8000`
- لوحة التحكم: `http://localhost:8000/admin`

## الإعداد (متغيرات بيئة اختيارية)

| المتغير | الغرض |
| --- | --- |
| `ADMIN_KEY` | كلمة مرور لوحة التحكم (أو ضعها في `admin-config.json`) |
| `SMTP_HOST`, `SMTP_USER`, `SMTP_PASS`, `ADMIN_EMAIL` | لتفعيل إشعار البريد عند كل طلب جديد |
| `PORT` | منفذ الخادم (الافتراضي 8000) |

## البنية

```
index.html      الموقع العام (HTML/CSS/JS بملف واحد)
admin.html      لوحة التحكم
serve.cjs       خادم Node: ملفات ثابتة + API (طلبات/ملفات/أعمال/إعدادات)
frames/         صور الهيرو السينمائي
logo.png        الشعار
```

## ملاحظات

- البيانات الحساسة وبيانات التشغيل **غير مرفوعة**: `admin-config.json`، `orders/`، `settings.json`، `work.json`، `work-images/` (انظر `.gitignore`).
- **كلمة مرور الإدارة والإيميلات المصرّح لها** توضع في `admin-config.json` (غير مرفوع). انسخ `admin-config.example.json` إليه واملأ القيم:
  ```json
  { "adminKey": "كلمة-مرور-قوية", "allowedEmails": ["a@example.com", "b@example.com"] }
  ```
- في الموقع العام، يُطابَق إيميل المالك عبر **بصمة SHA-256** (`OWNER_EMAIL_HASHES` في `index.html`) فلا تظهر الإيميلات في الكود. لتغيير المالكين، احسب بصمة الإيميل (sha256 بحروف صغيرة) وضعها هناك.
